# Picrochole

*Picrochole* est une simulation simple à l'échelon opérationnel de
conflits pré-napoléoniens et napoléoniens.

**Licence :** ce code est distribué sous la licence GNU GPL (cf. le
fichier [LICENSE](LICENSE)).
